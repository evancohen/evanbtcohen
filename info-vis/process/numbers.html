Another Numbers Display

I made some pretty significant changes based on the feedback that I recieved from my peers in terms of my number display. First off, I wasn't emulating my model enough, so I added dashes and fixed some more colors. I also made the names easier to read (human readable vs repository name). I also changed the data source to be something a little more meaninngful in terms of what it reperesents. I emulated my text model by using a tall font for the title, and smaller skinny font for the names and numbers. I played around with using some cursive, but that didn't fit very well.
