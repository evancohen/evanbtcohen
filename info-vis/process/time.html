Code Frequency

This display shows code contribution frequency over time. I used something similar to this for another display, which a stacked bar chart. This is very similar, but uses lines instead. It also uses green and red to indicate the addition and deletion of lines. I may end up chaning it to reflect my previous display (with two shades of green).
